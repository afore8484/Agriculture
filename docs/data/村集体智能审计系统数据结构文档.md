# 村集体智能审计系统数据结构文档

## 4. 数据需求

### 4.1 数据分类

#### 4.1.1 静态数据
- 行政区划、组织、用户、角色、字典、审计事项库、流程模板等主数据。

#### 4.1.2 动态数据
- 审计计划、方案、通知、承诺、资料调取、疑点、底稿、报告、复核、决定书、统计、日志、附件、流程实例等业务数据。

### 4.2 数据库设计概述

#### 4.2.1 数据库类型与选型
1. MySQL 8.x：核心事务数据存储。
2. Redis：高频字典和查询缓存。
3. MinIO/OSS/S3：附件对象存储。

#### 4.2.2 数据表结构
