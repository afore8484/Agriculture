## 一、公共与首页（8）

### 1. 获取行政区划树
- 方法：`GET`
- URL：`/api/nongjing/common/regions`
- 用途：获取行政区划层级树，用于全局筛选与级联选择。
- 入参：
  - parentCode：上级区划编码，可空
  - level：区划层级，可选（省/市/县/乡镇/村）
- 返回：
  - regionCode：区划编码
  - regionName：区划名称
  - level：区划层级
  - hasChildren：是否有子节点
- 数据来源：`dim_region`

### 2. 获取区划详情
- 方法：`GET`
- URL：`/api/nongjing/common/region-detail`
- 用途：获取单个区划的基础信息与 GIS 边界信息，用于地图定位。
- 入参：
  - regionCode：区划编码，必填
- 返回：
  - regionCode：区划编码
  - regionName：区划名称
  - parentRegionCode：上级区划编码
  - longitude：中心经度
  - latitude：中心纬度
  - geomWkt：边界几何
  - centroidWkt：中心点几何
- 数据来源：`dim_region`

### 3. 获取时间维度选项
- 方法：`GET`
- URL：`/api/nongjing/common/time-dimensions`
- 用途：获取月/季/年等时间维度下拉选项。
- 入参：
  - year：年份，可选
  - type：返回类型（月/季度/年度），可选
- 返回：
  - timeKey：时间键
  - calendarDate：日期
  - yearNo：年
  - quarterNo：季度
  - monthCode：月份编码
  - label：展示文本
- 数据来源：`dim_time`

### 4. 获取指标字典
- 方法：`GET`
- URL：`/api/nongjing/common/metrics`
- 用途：获取页面指标字典，供图表切换与指标统一口径使用。
- 入参：
  - moduleCode：模块编码，可选

  - metricDesc：指标说明
- 数据来源：`dim_metric`
