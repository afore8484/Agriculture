# 村委财务事务管理系统数据结构文档

# 一、公共与首页（10）

------

## 1. 获取行政区划树

- 方法：`GET`
- URL：`/api/nongjing/common/regions`
- 用途：获取行政区划层级树，用于全局筛选、级联选择、权限过滤。
- 入参：
  - parentCode：上级区划编码，可空
  - level：区划层级（省/市/县/乡镇/村），可选
- 返回：
  - regionCode：区划编码
  - regionName：区划名称
  - level：区划层级
  - hasChildren：是否存在子节点
- 数据来源：`dim_region`

------

## 2. 获取首页统计指标

- 方法：`GET`
- URL：`/api/home/stats`
- 用途：首页经营与财务关键指标展示。
- 入参：
  - orgId：主体ID
  - period：期间
- 返回：
  - memberCount：成员数量
  - capitalAmount：出资金额
  - salesAmount：销售额
  - expenseAmount：支出金额
  - projectCount：项目数量
- 数据来源：统计视图

------
