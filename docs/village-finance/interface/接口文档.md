## 工作台接口补充（2026-03-21）

说明：以下为当前后端已实现的工作台接口口径；若与上方旧表述冲突，以本节为准。

### 兼容路径
1. `/api/home/stats` <=> `/api/village-finance/home/stats`
2. `/api/home/todos` <=> `/api/village-finance/home/todos`
3. `/api/home/charts` <=> `/api/village-finance/home/charts`
4. `/api/home/progress` <=> `/api/village-finance/home/progress`

### 新增能力（工作台）
1. `GET /api/village-finance/home/asset-distribution`
2. `GET /api/village-finance/home/recent-vouchers`

### 字段口径（当前实现）
1. `GET /api/village-finance/home/stats`
   - 入参：`ledgerId?`、`orgId?`、`periodId?`、`period?(yyyy-MM)`、`userId?`
   - 出参：`ledgerId`、`periodId`、`incomeAmount`、`expenseAmount`、`assetAmount`、`pendingTodoCount`、`urgentTodoCount`
2. `GET /api/village-finance/home/todos`
   - 入参：`userId`(必填)、`limit?`
   - 出参：`todoId`、`todoType`、`title`、`createTime`、`taskId`、`instanceId`、`bizType`、`bizId`、`nodeName`、`taskStatus`、`urgency`
3. `GET /api/village-finance/home/charts`
   - 入参：`ledgerId?`、`orgId?`、`periodId?`、`period?(yyyy-MM)`、`months?`
   - 出参：`periodLabel`、`incomeAmount`、`expenseAmount`
4. `GET /api/village-finance/home/asset-distribution`
   - 入参：`ledgerId?`、`orgId?`
   - 出参：`categoryName`、`amount`
5. `GET /api/village-finance/home/recent-vouchers`
   - 入参：`ledgerId?`、`orgId?`、`periodId?`、`period?(yyyy-MM)`、`limit?`
   - 出参：`voucherId`、`voucherNo`、`voucherDate`、`summary`、`amount`、`direction`、`auditStatus`、`voucherType`、`bizId`
6. `GET /api/village-finance/home/progress`
   - 入参：`ledgerId?`、`orgId?`、`periodId?`、`period?(yyyy-MM)`
   - 出参：`voucherTotal`、`auditedTotal`、`percent`

### 关联补充
- `GET /api/village-finance/vouchers` 已补 `summary` 字段。
