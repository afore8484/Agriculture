# S3-P1-A 资产管理 R1-A（最小管理包）

- 时间：2026-03-18
- 阶段：S3-P1-A
- 范围：资产分类与资产卡片最小管理，不扩展折旧/处置/盘点/资产结账

## 1. 本轮范围

1. 资产分类：新增、修改、查询、删除受限。
2. 资产卡片：新增、修改、详情、列表、删除受限。
3. 严格不做折旧、处置、盘点、资产结账/反结账、自动制证。

## 2. 已完成

1. 数据落地（Flyway）：
   - 新增 `V6__asset_r1a_minimal.sql`。
   - 新建核心表：
     - `fin_asset_category`
     - `fin_asset_card`
   - 新建删除保护关联表（最小结构）：
     - `fin_asset_depreciation`
     - `fin_asset_disposal`
     - `fin_asset_inventory`
     - `fin_asset_inventory_detail`
2. 资产服务：
   - 新增 `FinanceAssetService` / `FinanceAssetServiceImpl`。
3. 资产接口：
   - 新增 `AssetController`，兼容 `/api/village-finance/asset` 与 `/api/asset`。
   - 分类接口：`/category/create`、`/category/update`、`/category/list`、`/category/delete`。
   - 卡片接口：`/card/create`、`/card/update`、`/card/detail`、`/card/page`、`/card/delete`。
4. 删除保护：
   - 分类存在卡片引用时拒绝删除；
   - 资产发生折旧/处置/盘点明细时拒绝删除；
   - 资产已按统一来源模型关联凭证时拒绝删除。
