# S3-P2-A 合同管理 R1-A 最小管理包

## 1. 本轮范围
1. 合同主档：新增、修改、分页、详情、删除受限（逻辑删除）。
2. 合同变更：新增、列表查询。
3. 合同验收：新增、列表查询。
4. 合同终止 / 续签：新增、列表查询。
5. 附件 / 操作日志：附件上传+查询、操作日志查询最小留痕。
6. 不实现：审批联动、收付款、资金/凭证联动。

## 2. 落地内容
1. Flyway：
   - 复用 `V7__contract_r1a_minimal.sql`，本轮无新增 migration。
2. 服务与接口：
   - 新增 `FinanceContractService` / `FinanceContractServiceImpl`。
   - 新增 `ContractController`，兼容 `/api/village-finance/contracts` 与 `/api/contract`。
3. 最小规则：
   - `contractNo` 唯一。
   - 金额字段非负校验。
   - 删除保护：存在变更/验收/终止/续签/附件记录时拒绝。
   - 删除策略：逻辑删除（`status=DELETED`）。
   - 留痕：主档与生命周期操作写入 `fin_contract_operation_log`。
