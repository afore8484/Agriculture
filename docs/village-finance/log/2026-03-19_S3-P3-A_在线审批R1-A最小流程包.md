# S3-P3-A 阶段日志（在线审批 R1-A 最小流程包）

1. 流程定义：新增、修改、查询、启停用。
2. 流程实例：发起、详情、列表。
3. 审批任务：待办、已办、通过/驳回处理。

4. 流程形态：最小单节点流转，处理后实例结束。
