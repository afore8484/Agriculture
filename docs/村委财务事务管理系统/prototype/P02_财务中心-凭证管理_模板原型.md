# 村委财务事务管理系统 - P02 财务中心-凭证管理模板化原型说明

- 系统名称：村委财务事务管理系统
- 页面编号：P02
- 页面名称：财务中心-凭证管理
- 所属模块：财务中心

## 1. 页面布局说明
- 页面目标：用于财务中心-凭证管理的业务处理，覆盖记账凭证录入、修改、审核、反审核、导出、附件管理、回收站等功能。
- 页面功能：
  - 新增凭证
  - 保存并新增
  - 上传附件
  - 查看附单据数量
  - 打印设置
  - 打印凭证
  - 审核
  - 反审核
  - 导出凭证
  - 修改凭证
  - 插入凭证
  - 查询凭证
  - 回收站查询
  - 还原已删除凭证
  - 彻底删除凭证
- 布局来源说明：查询区、凭证编辑表单、分录表格、附件区、操作按钮区、回收站列表

## 2. 页面结构
1. 页面头部：页面标题 + 页面编号
2. 查询区：凭证编号、日期范围、会计科目、金额范围、审核状态
3. 操作区：新增、导出、打印、审核、反审核
4. 表格区：凭证列表
5. 编辑区：凭证表单 + 分录表格 + 附件上传
6. 回收站区：已删除凭证列表

## 3. 关键字段和操作按钮
### 3.1 关键字段
- 凭证编号
- 制单日期
- 摘要
- 会计科目
- 借方金额
- 贷方金额
- 备注
- 附件
- 审核状态
