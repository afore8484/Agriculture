# 农经一张图前端

本工程是“农经一张图”模块的正式前端代码，目录固定为：
`E:\workspaces\Agriculture\prototype\modules\nongjing-map\frontend`

## 启动方式

1. 执行 `npm install`
2. 执行 `npm run dev`
3. 执行 `npm run build`
4. 执行 `npm run test`

## 环境变量

- `VITE_API_BASE_URL`：后端服务地址，默认走 service 层合同样例
- `VITE_ENABLE_API_PANEL`：是否显示接口调试面板，默认关闭
- `VITE_AUTH_TOKEN`：Bearer Token，可选
