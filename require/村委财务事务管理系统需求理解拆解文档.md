# 二、系统总体理解

本系统主要用于 **村集体财务与事务管理信息化**，核心目标是：

1. 规范村集体财务管理流程
2. 提供完整财务账务处理能力
3. 实现资产、合同等业务数据统一管理
4. 提供在线审批流程管理
5. 支持银行直联与资金监管
6. 生成规范财务报表
7. 提供审计与监管依据

系统核心业务围绕 **资金、资产、合同、审批、银行交易** 展开。

# 五、模块需求理解

## 1 财务中心

### 模块定位

负责 **村集体财务日常记账与资金管理**。

核心功能：

- 出纳管理
- 凭证管理
- 财务账簿
- 资金报表
- 月末结账

------

### 1.1 出纳管理

主要管理：

- 银行日记账
- 现金日记账
- 内部转账
- 账户管理

主要功能：

| 功能         | 说明           |
| ------------ | -------------- |
| 日记账登记   | 记录资金收支   |
| 凭证自动生成 | 日记账生成凭证 |
| 模板导入     | 批量导入数据   |
| 数据导出     | Excel/PDF      |
| 查询筛选     | 多条件查询     |

例如：

- 查询某时间段银行凭证
- 按金额范围查询
- 按交易对象查询

这些能力用于 **财务核对与资金追踪**。

------

### 1.2 凭证管理

系统支持完整的凭证生命周期：

1. 凭证录入
2. 凭证修改
3. 凭证审核
4. 凭证反审核
5. 凭证导出
6. 凭证删除
7. 凭证回收站

凭证信息包括：

- 摘要
- 会计科目
- 借方金额
- 贷方金额
- 附件

凭证支持 **附件上传（发票、合同等）**。

------

### 1.3 账簿管理

系统支持多种账簿：

| 账簿类型   |
| ---------- |
| 序时账     |
| 科目余额表 |
| 科目汇总表 |
| 明细账     |
| 总账       |
| 总账日记账 |
| 收支明细表 |
| 辅助核算账 |

账簿主要功能：

- 查询
- 穿透凭证
- 导出
- 打印

例如：

用户可从 **序时账直接查看凭证详情**。

------

### 1.4 月末处理

系统支持：

- 损益结转
- 月末结账
- 反结账
- 结账试算

结账流程：

```
期末处理
  ↓
结转损益
  ↓
试算
  ↓
月结
```

------

## 2 报表中心

负责生成 **村集体财务报表**。

主要报表：

| 报表             |
| ---------------- |
| 资产负债表       |

| 科目汇总表       |

例如：

- 资产结构分析

- 趋势分析

------

## 3 资产管理

### 模块定位

管理 **村集体固定资产**。

资产信息包括：

- 资产编号
- 资产名称
- 型号
- 入账日期
- 原值

资产生命周期：

```
资产新增
   ↓
资产变更
   ↓
资产折旧
   ↓
资产处置
```

系统支持：

- 自动折旧计算
- 一键生成折旧凭证
- 资产结账

------

## 4 合同管理

用于管理 **村集体合同业务**。

| 字段     |

| 合同金额 |
| 合同期限 |
